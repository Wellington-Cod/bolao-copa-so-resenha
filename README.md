# Release 1.0.0 - Bolão da Copa Só Resenha

Data da Release: 18/06/2026

## Funcionalidades

* Cadastro de participantes
* Controle de pagamentos
* Ranking automático
* Pódio dos líderes
* Seleção do campeão da Copa
* Cadastro e edição de jogos
* Cadastro de resultados
* Cálculo automático de pontuação
* Bandeiras dos países
* Exportação pública do bolão
* Visualização responsiva para celular
* Organização de jogos pendentes e concluídos
* PIX para inscrição com botão copiar

## Critérios de Pontuação

* Acerto do vencedor: 3 pontos
* Acerto do placar exato: 6 pontos
* Acerto do campeão: 9 pontos

## Critério de Desempate

1. Maior número de placares exatos
2. Acerto do campeão
3. Divisão do prêmio ou decisão entre os participantes
